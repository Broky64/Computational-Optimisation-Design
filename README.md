# Computational Optimisation Design

This repository contains materials, code, and assignments for the Computational Optimisation Design course.

## Project Structure

- `src/`: Source code for optimization algorithms and models.
- `notebooks/`: Jupyter notebooks for experimentation and visualization.
- `docs/`: Course documentation and reports.
- `data/`: Input datasets.
- `results/`: Generated output, plots, and results.
- `tests/`: Unit tests for the codebase.

## Getting Started

### Prerequisites

- Python 3.x
- Recommended: Virtual environment (venv or conda)

### Installation

```bash
pip install -r requirements.txt
```
